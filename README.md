# Student Performance — Data Cleaning, EDA, and Insights
## Project Overview
This project analyzes a student performance dataset to understand how
study habits, attendance, tutoring, and other factors relate to exam
performance.
## Tools Used
- Python
- Pandas
- Matplotlib
## Key Tasks Completed
- Data cleaning (missing values, messy categories, outliers)
- Feature engineering (study level, attendance level, improvement gap)
- Exploratory Data Analysis (EDA)
- Visualizations and insights
## Files in This Repository
- `Month1_StudentPerformance_YourName.ipynb` → main notebook
- `student_performance_clean.csv` → cleaned dataset
- `Month1_Summary_YourName.pdf` → 1-page summary report
